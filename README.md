# House-Price-Prediction
Problem Statement

Predict house prices based on property features such as house size, number of bedrooms, and location score.

Dataset Description

The dataset contains 15 records with the following features:

Size_sqft

Bedrooms

Location_score

Price (Target variable)

Model Used

Linear Regression

Results

The model was evaluated using:

Mean Absolute Error (MAE)

R² Score

The model demonstrated good predictive ability, especially when additional meaningful features such as price per square foot were included.

Conclusion

House size and location significantly influence house prices. Feature engineering helped improve the model performance and demonstrated the importance of selecting meaningful variables.
